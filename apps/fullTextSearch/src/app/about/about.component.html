Full Text Search Demo
  • Setup and Code  explains how to create a SingleStoreDB Cloud for this demo.
    • Go to the Setup Tab, key in Database URL, UserName and Password fields.
      • Optionally you can change the database name.
      • Please leave the table name and column name as defaulted, unless you are planning to load and use your own data.
    • Press the Test and Save button. If the details were entered correctly and the database is running, it will display "Connection successful and settings are saved."
    • If the table does not exist it will have a Create All button, which will require the following steps.
      • Press the Create All button an it will create the database (if not present), the table, and a pipeline to load the table.
      • Please be patient. Within a minute a status bar for the data load will appear.
      • Searching on partial loads will work, but some results will be missing until it reaches 100%.
  • Go to the Search Tab and test the searching.
    Example Searches
    • Like Search
      • %haggle silent%
      • %haggle silent%special requests%bold pinto%
    • Match Search
      • +"haggle silent" +"special requests" +"bold pinto"
      • +"haggle silent"
      • hagle~ AND scilent~
    • Highlight and Match Search
      • +"haggle silent" +"special requests" +"bold pinto"
    Note: The API calls are visible in the network section of the browser developer toolkit; F12 starts this tool in Chrome.
